# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-02-04

### Added
- Initial crosswalk mapping between AIUC-1 v1 and ISO/IEC 42001:2023
- 78 control mappings across all 6 AIUC-1 domains
- Coverage of ISO 42001 Clauses 4-10 and Annex A controls
- Domain mappings:
  - Domain A (Data & Privacy) → Annex A.5, Clause 8
  - Domain B (Security) → Annex A.6, Annex A.8
  - Domain C (Safety) → Annex A.9, Clause 6
  - Domain D (Reliability) → Annex A.7, Clause 9
  - Domain E (Accountability) → Clauses 5-10, Annex A.2, A.3
  - Domain F (Society) → Annex A.4, A.10
