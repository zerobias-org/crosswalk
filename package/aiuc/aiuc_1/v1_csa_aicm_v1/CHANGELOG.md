# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-02-04

### Added
- Initial crosswalk mapping between AIUC-1 v1 and CSA AI Controls Matrix v1
- 76 control mappings across all 6 AIUC-1 domains
- Coverage of 18 CSA AICM security domains
- Domain mappings:
  - Domain A (Data & Privacy) → DSP, MDS
  - Domain B (Security) → AIS, IAM, CEK, TVM
  - Domain C (Safety) → AIS, SEF, GRC
  - Domain D (Reliability) → BCR, MDS
  - Domain E (Accountability) → GRC, A&A, STA
  - Domain F (Society) → GRC
